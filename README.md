# Seniors' Integration Program Website

A cinematic, no-scroll SIP website built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- Centered 2x2 landing grid
- In-place sliding section transitions
- Lightweight visual treatment for faster loading
- Single-page interaction model

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Development
```powershell
npm install
npm run dev
```

## Build
```powershell
npm run build
npm run start
```
